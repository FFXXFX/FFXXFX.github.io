es6新语法

作用域

  • 除了es5的全局作用域、局部作用域和eval作用域外，es6拥有块级作用域

  • 代码块中使用特定关键字(如let，const)定义的变量，只在代码块中有效，在代码块外访问不到

  • 块级作用域同样符合作用域链

关键字

let

  • 在同一级作用域中变量不能重复声明
  • 在块级作用域中有效
  • 不存在变量提升
  • 定义的全局变量不会绑定给window对象

const常量

  • 在同一级作用域中变量不能重复声明
  • 需要赋初值
  • 常量值不能修改
    • 如果是基础数据类型，则不能修改值
    • 如果是引用数据类型，则不能修改指向，但能修改引用数据类型里的内容
  • 在块级作用域中有效
  • 不存在变量提升
  • 定义的全局变量不会绑定给window对象
  • 一般常量名全部使用大写

  • 模板字符串可使用${表达式}来显示表达式的值

对象的简化写法

  • 当用一变量作为对象的属性值，而且变量名与属性名相同时，可省略属性名

箭头函数

  • 格式为( 参数列表 ) => { 函数体 }
  • 只能使用函数表达式来创建函数，使用不了函数声明进行创建
  • this指向是静态的，指向声明时所在的作用域下的this(箭头函数没有this，所以沿着作用域链去找)

  • 箭头函数没有显式原型prototype，故不能作为构造函数

  • 没有arguments属性

  • 特定情况下可进行缩写

    • 当参数列表有且仅有一个参数时，可省略小括号，格式为参数 => { 函数体 }
    • 当函数体只有一条return语句时，可省略return和花括号，格式为(参数列表) => 返回值
    • 上述可同时省略，格式为参数 => 返回值

  • 箭头函数适合与this无关的回调函数，如定时器

  • 箭头函数不适合与this有关的回调函数，如事件监听

参数赋初值

  • 赋初值的参数要在参数列表的最后
  • 参数赋初值可与解构赋值一同使用

rest参数

  • 引入rest来代替arguments
  • rest参数必须放在参数列表的最后

扩展运算符

  • ...扩展运算符可将数组和伪数组转化为以,分隔的参数序列
  • 作用
    • 数组的合并
    • 数组的克隆
    • 伪数组转化为真实的数组

Symbol

  • 一种全新的基础数据类型

  • Symbol值是唯一的，表示独一无二的，用来解决命名冲突的问题

  • Symbol.for(desc)表示从Symbol注册表中读取描述为desc的Symbol，如果没有，则会创建此描述的Symbol，并加入Symbol注册表中(其他方式创建的Symbol不会加入注册表)

  • Symbol值不能进行运算

  • 对象的Symbol类型的属性不会被for ... in和for...of所遍历到

  • 在使用Symbol作为对象属性名时需要使用[]括起来

迭代器（Iterator）

  • es6提供了新的遍历方式for...of

    • for(let key in Array){}的key是键
    • for(let value of Array){}的value是值

  • 只有拥有迭代器接口（[Symbol.iterator]属性）的对象，才能使用for...of进行遍历，因为使用for...of时，会调用迭代器接口

  • 迭代器接口的会创建一个指向起始位置的指针，并返回一个具有next方法的对象

  • next方法会返回一个具有value属性和done属性的对象，并且每次调用都会使指针后移，value属性保存的是值，done代表是否索引结束

  • for(let value of arr){}的工作流程

    • 调用arr.[Symbol.iterator]()，创建了指向起始位置的指针，并且得到next函数
    • 调用next的函数，指针后移，返回一个具有value属性和done属性的对象
      • 如果done为true，代表迭代结束，for(let value of arr){}运行完成
      • 否则，将对象的value传给for(let value of arr){}进行使用，并再次调用next函数

生成器函数

  • 异步编程的解决方案，避免回调地狱
  • 生成器函数本质为生成器构造函数，每次调用时都会返回一个生成器对象
  • 生成器函数在function关键字后有一个*，函数体有若干条yield语句用于暂停生成器的执行
  • 调用生成器对象的next方法，会把next方法传进的参数赋值给yield语句，然后会执行下一条yield之前的代码和yield之后的表达式，并返回一个value属性值为表达式的结果和done属性的对象
  • 使用yield*可实现生成器嵌套

Promise

  • 异步编程的解决方案，避免回调地狱
